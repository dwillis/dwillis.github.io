Failure Patterns

False Negatives

Over-Extraction

Including extra text like "Not authorized by any candidate"

Partial Extraction

Truncated names (e.g., "Pennsylvania Democratic" vs. full committee name)

Format Violations

Invalid JSON or malformed output ‚Äî common in models without structured output support

Key Finding

Model size matters: Large models (30B+ params) show subtle errors; smaller models (<7B) exhibit severe failures including complete task misunderstanding
