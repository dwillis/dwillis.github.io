Staying on Offense in the Big Ten

The Big Ten had some of the nation’s more prolific offenses last season. I’m going to make a bar chart of the teams’ average number of offensive plays over the course of the season so I can see which team was lining up the most on average. To do that, I’ll need the game stats from last season and will need to focus on just the Big Ten schools. Fortunately the chapter on clustering already has a list I can use to filter.

big10 <- c("Nebraska Cornhuskers", "Iowa Hawkeyes", "Minnesota Golden Gophers", "Illinois Fighting Illini", "Northwestern Wildcats", "Wisconsin Badgers", "Indiana Hoosiers", "Purdue Boilermakers", "Ohio State Buckeyes", "Michigan Wolverines", "Michigan State Spartans", "Penn State Nittany Lions", "Rutgers Scarlet Knights", "Maryland Terrapins")
  
big_ten_logs <- logs %>% filter(TeamFull %in% big10)

Then I’ll need to calculate an average number of offensive plays for each team, collapsing the game-by-game data into a single row for each team.

plays_average <- big_ten_logs %>% 
  group_by(TeamFull) %>% 
  summarise(average_plays = mean(OffensivePlays)) %>% 
  arrange(desc(average_plays))

Now I’ll create a bar chart of these averages:

ggplot() + 
  geom_bar(
    data=plays_average, 
    aes(x=TeamFull, weight=average_plays)
  )

I can’t read the team labels, so I’ll need to switch from vertical bars to horizontal ones using coord_flip and reorder it so that the team with the highest average is first using reorder. Then I’ll add sensible labels.

ggplot() + 
  geom_bar(
    data=plays_average, 
    aes(x=reorder(TeamFull, average_plays), weight=average_plays)
  ) +
  coord_flip() +
  labs(
    title="Average Offensive Plays Among Big Ten Teams", 
    x="Team", 
    y="Plays")

Analysis

Maryland was second in the conference last season in average number of offensive plays per game behind Purdue and just ahead of Penn State. Ohio State, which had an explosive offense, ranked fourth, although that could be because it scored quickly and not on lengthy drives. That’s worth checking out further.

One thing that surprised me about this chart is the relatively small gap between the team with the highest average and the team with the lowest average. Iowa makes sense as the team with the smallest average, but on average it only ran eight fewer plays a game than Purdue did.
